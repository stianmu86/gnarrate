# Narrate

A premium personal audiobook app that converts articles, PDFs, and notes into high-quality AI-narrated audio. Built with a warm, Nordic-inspired design philosophy.

## Stack

- **Frontend**: Expo / React Native (iOS + Web) with NativeWind
- **Backend**: Supabase (Auth, PostgreSQL, Storage, Edge Functions)
- **AI/GPU**: Modal.com (TTS inference, LLM enrichment)
- **Payments**: Stripe

## Getting Started

```bash
npm install
npx expo start
```

See the implementation roadmap docs for detailed setup instructions.
